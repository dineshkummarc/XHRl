I've set the delay on the jQuery library to 5 seconds so a timeout will be caused!

If debugging isn't set to true then remember to check: xhrl.errors via the console

If you want to see the script loading working then just open jquery.php and change sleep(5); to sleep(4); so jQuery loads before timeout is reached.
